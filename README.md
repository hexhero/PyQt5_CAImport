CA证书数据导入程序
-----------------

* pyqt5 
* Qt Designer (pyuic5.exe -o frame.py .\untitled.ui) UI转py
* pyinstaller (pyinstaller.exe -F .\main.py  -w -n yangb -i .\app.ico) 打包.exe可执行程序 
    * -F 产生成一个文件 
    * -D 产生成一个目录
    * -w: windows窗口 
    * -c 命令行窗口 
    * -i 图标 
    * -n 名称
